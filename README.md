# Burj Al Arab

A Single page web app with dashboard  where people can pick the date and time and fix room booking.

[Live Website](https://burj-al-arab-59182.web.app/) 

### Features
* Online Booking making
* Clients Database

### Front-end Thechnology : 
* React.js
* Bootstrap4
* CSS3

### Back-end Technology
* Node.js
* Express.js
* MongoDB
